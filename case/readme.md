# Create the case

## Install

### required

- install python (3.12)
- pip install build123d  # 0.10.0

### optional for viewing

- pip install ocp_vscode  # 3.1.2
- install visual studio code  (1.111)
- install OCP CAD Viewer plug in (3.1.2)

the versions in paranthesis or comments are my versions. Newer should work also, but are not tested.

## Start

### with viewing

- open Visual Studio Code
- start one of the main_*.py files

### only creating *.stl files

- start one of the main_*.py files

## Results

- the STL file should be shown in the OCP Viewer
- the STL file should be stored in the case/#output directory
